# telegram-claude-bot (Claude Skill)

A Claude Code skill that sets up a production-ready Telegram bot bridging your Telegram messages directly into an active Claude Code session via `claude --resume`.

## What it does

- Chat with your Claude Code session from Telegram (phone or desktop)
- Full session continuity — Claude remembers everything from the current session
- Auto-detects new sessions when you `/clear` in Claude Code
- Handles crashes, restarts, and Windows auto-start

## Features

- Single worker thread (no concurrent `--resume` conflicts)
- HTML rendering (Claude markdown renders properly in Telegram)
- UTF-8 safe (no garbled characters)
- Typing indicator + edit-in-place responses
- Long responses (>1500 chars) sent as `.txt` file attachments
- `/help` `/status` `/session` `/cancel` commands
- Lock file (prevents two bot instances)
- Startup + crash notifications to your Telegram
- Graceful shutdown
- Watchdog process with auto-restart
- Windows Scheduled Task for boot auto-start

## Installation

### 1. Clone and copy skill

```bash
git clone https://github.com/evanlee-gr/claude-skills
```

Copy `telegram-claude-bot/` into your Claude skills directory:

**Windows path:**
```
%APPDATA%\Claude\local-agent-mode-sessions\skills-plugin\<plugin-id>\<session-id>\skills\
```

Find your plugin/session IDs by browsing to that path — there will be one subfolder each.

### 2. Register in manifest.json

Add this entry to the `skills` array in `manifest.json` (same directory as the `skills/` folder):

```json
{
  "skillId": "telegram-claude-bot",
  "name": "telegram-claude-bot",
  "description": "Sets up a production-ready Telegram bot that bridges Telegram messages directly into a Claude Code session using claude --resume. Use whenever the user wants to control Claude Code from Telegram, set up a Telegram bot for Claude, or access Claude Code from their phone.",
  "creatorType": "user",
  "updatedAt": "2026-04-05T00:00:00.000Z",
  "enabled": true
}
```

### 3. Restart Claude Code

The skill will now be available. Trigger it by saying:
> *"set up a telegram bot for Claude Code"*

## Requirements

- Python 3.8+
- `pip install requests psutil`
- Claude Code with `claude` CLI in PATH
- Telegram account + bot token from @BotFather
- Windows (for auto-start feature; bot itself works on any OS)

## Files generated by the skill

| File | Purpose |
|------|---------|
| `telegram_claude_bot.py` | Main bot |
| `telegram_bot_watchdog.py` | Crash-restart watchdog |
| `setup_telegram_autostart.py` | Windows Scheduled Task setup |

## Bot commands

| Command | Description |
|---------|-------------|
| `/help` | List all commands |
| `/status` | Uptime, session ID, queue depth, session size |
| `/session` | Show active session ID |
| `/cancel` | Abort the current Claude task |
